Mermaid Flow Player includes features for animating diagrams.

## Core Features

- **[Web Component](installation)** — `<mermaid-flow-player>` with declarative attributes
- **[Kitchen Sink Showcase](features/showcase)** — Full feature gallery with copy-paste examples
- **[Auto Modes](features/auto-modes)** — Zero-config usage with 4 progressive enhancement layers
- **[Animations & Easing](features/animations)** — 30+ easing functions
- **[Edge Detection](features/edge-detection)** — Multi-strategy edge animation system
- **[Interactive Mode](features/interactive)** — Step-through diagrams with path selection
- **[Narration](features/narration)** — Auto-updating narration text during playback

## Builder & Composition

- **[Scenario Builder](features/builder-api)** — Fluent chainable API for building scenarios
- **[Plugins](features/plugins)** — Build extensions with lifecycle hooks

## Configuration

- **[Themes](features/themes)** — Light, dark, and auto themes with persistence
- **[Query Parameters](features/query-params)** — Configure via URL parameters

## Diagram Types

All features work with 7 diagram types:
[Flowcharts](diagram-types/flowcharts), [Sequence](diagram-types/sequence), [State](diagram-types/state), [Gantt](diagram-types/gantt), [Journey](diagram-types/journey), [Class](diagram-types/class), and [ER](diagram-types/er) diagrams.
